Context

Processing has been a very nice effort to make all this java swing nonsense accesible to anyone. Kudos to the team, honestly. The editing software, though, I have found lacking and frustrating. I found this as something quite normal since it's an open source initiattive and priorities have been elsewhere (ie make the library more robust than implement nice-to-have features for the editor).

Now imagine you can harvest botht the powers given to you by the Processing library and an awesome IDE (in this case I've chosen IntelliJ community eddition) using the processing library in a gradle project.
